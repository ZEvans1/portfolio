Wikipedia Homepage

The purpose of this project was to emulate the front page of Wikipedia as closely as possible (not including functionality). This project essentially reviewed all of the content that was covered during the week.

Uses:

  • HTML
  • CSS
  • Bootstrap

Vacation

The focus of the vacation project was to create a page for a vacation destination and to begin using Bootstrap. Specific examples include: Jumbotron, blockquotes, and helper classes.

Uses:

  • HTML
  • CSS
  • Bootstrap

Interior Decorator

A page for a fictional interior decorator. This page was made to practice using floats. It also was made to practice using media queries. The page will change as you adjust the width of the page.

Uses:

  • HTML
  • CSS

Animal Shelter

This page about an animal shelter was made to practice cascading styles and to understand the concept of inheritance.

Uses:

  • HTML
  • CSS

Blog Page

A blog page about whiteboards. This project served as an introduction to using div tags.

Uses:

  • CSS

Band Page

A page for the band Animals as Leaders. Floats were used for this time during this project. I don't think I quite had a full handle on them while it was made because the sidebar is having an impact on the image spacing. This project also focused on implementing the box model.

  • CSS

Cupcake Shop
